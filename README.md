#The Pink Crusade

Pink crusade app with drag and drop dedications for SBMC

###Getting Started
Set it up!
```
git clone https://github.com/dannyvassallo/thepinkcrusade.git
cd thepinkcrusade
bundle install
rake db:migrate
rails s
```

###Embed The Widget
```
<iframe width="810px" height="950px" src="..."></iframe>
```
